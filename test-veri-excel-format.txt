Excel'den Veri Yükleme Test Dosyası
=====================================

Bu dosyayı Excel'de açıp, aşağıdaki verileri kopyalayıp AdminDashboard'daki "Excel'den Veri Yükle / Güncelle" butonuna tıklayıp yapıştırabilirsiniz.

Beklenen Format:
Hesap Kodu | Hesap Adı | Borç | Alacak | Borç Bakiyesi | Alacak Bakiyesi

Örnek Veriler (Tab ile ayrılmış - Excel'den kopyalarken otomatik olarak tab karakteri kopyalanır):

131.001.001	NAMIK KETHÜDA	38922.78	40374.64	0	1451.86
131.001.002	SELVET DURUR	37027.22	42792.89	0	5765.67
131.001.003	MELİS YILDIZ	31676.64	34737.71	0	3061.07
131.001.004	YILDIZ ÇALIŞIR ( SİBEL KIZIL)	28085.12	22216.88	5868.24	0
131.001.005	UĞUR ERAVCI	48696.54	46000.00	2696.54	0
131.001.006	HAMDİYE ORTA	38178.05	38685.10	0	507.05
131.001.007	SANİYE KIRTI	36022.86	37341.00	0	1318.14
131.001.008	ÖNDER YAYA (GAZİ AYGÜNEŞ ALTUNHAN)	36271.70	27425.00	8846.70	0
131.001.009	AYŞE MERCAN YAĞCIOĞLU	33118.63	35927.68	0	2809.05
131.001.010	BAHRİ GÜN	43627.16	33349.00	10278.16	0

Kullanım Talimatları:
1. Excel'de yeni bir dosya açın
2. Aşağıdaki başlıkları ilk satıra yazın (opsiyonel - başlıklar otomatik atlanır):
   Hesap Kodu	Hesap Adı	Borç	Alacak	Borç Bakiyesi	Alacak Bakiyesi
3. Verileri alt satırlara yapıştırın veya yazın
4. Tüm verileri seçin (Ctrl+A veya fare ile)
5. Kopyalayın (Ctrl+C)
6. AdminDashboard'da "Excel'den Veri Yükle / Güncelle" butonuna tıklayın
7. Açılan modal'daki textarea'ya yapıştırın (Ctrl+V)
8. "Verileri İşle ve Kaydet" butonuna tıklayın

Not: 
- Sayılar nokta (.) ile ondalık ayırıcı olarak kullanılmalı (örn: 38922.78)
- Binlik ayırıcı kullanmayın (örn: 38.922,78 yerine 38922.78)
- Tab karakteri ile ayrılmış olmalı (Excel'den kopyalarken otomatik)
